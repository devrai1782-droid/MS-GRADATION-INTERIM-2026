📌 इस System में क्या-क्या होता है:
✅शिक्षकों का ग्रेडेशन डेटा — सभी JD/DEO के माध्यमिक शिक्षकों की वरीयता सूची
✅Record Edit करना — DPI/JD/DEO अपने अधिकार के अनुसार जानकारी update कर सकते हैं
✅PDF Print — किसी भी record का PDF निकाल सकते हैं
✅Summary Report — जिलेवार/संभागवार summary देख सकते हैं
✅Document Upload — Transfer order, NOC आदि Cloudinary पर upload कर सकते हैं
✅Real-time Collaboration — एक साथ कई users data देख व edit कर सकते हैं
⚠️ महत्वपूर्ण नियम:
• JD अपने संभाग के माध्यमिक शिक्षकों के records ही edit कर सकते हैं
• DEO अपने जिले के records ही edit कर सकते हैं
• DPI सभी records देख व edit कर सकते हैं
• Password किसी को share न करें

🔐 System में Login कैसे करें

1
User ID डालें
अपना User ID डालें — जैसे JDBHOPAL, DEOBHOPAL, DPI
User ID अपने आप CAPITAL हो जाएगी
2
Password डालें
DPI द्वारा दिया गया password डालें। भूल जाने पर DPI से संपर्क करें।
3
ACCESS SYSTEM दबाएं
Button दबाएं या Enter key press करें। Supabase Cloud से data load होगा।
4
Data Load होगा
Login के बाद Supabase से automatically data load होगा। Progress bar दिखेगी।
❌ Login Error आए तो:
• User ID या Password गलत है — दोबारा check करें
• DPI से नया password लें
• Internet connection check करें

📋 Records कैसे देखें

📊 मुख्य Table
Login के बाद सभी माध्यमिक शिक्षकों की list table में दिखेगी। हर row एक शिक्षक का record है। Scroll करके सभी columns देखें।
🖱️ Record Details देखना
SHOW FORM बटन पर Click कर नीचे search Box में Unique ID टाइप कर Search करें। उस शिक्षक की पूरी जानकारी FORM में खुलेगी।
📈 Summary देखना
Toolbar में "SUMMARY" button दबाएं → जिलेवार/संभागवार statistics और charts दिखेंगे।
📜 History देखना
"HISTORY" button → किसी record में क्या-क्या बदलाव हुए, कब और किसने किए — सब दिखेगा।

✏️ Record Edit कैसे करें

1
Record चुनें
SHOW FORM खोलें → search box में Unique ID डालकर Search करें।
2
Form खुलेगी
शिक्षक की पूरी जानकारी form में दिखेगी। White fields edit हो सकती हैं, Grey fields locked हैं।
3
जानकारी बदलें
जो field बदलनी हो उसमें नई जानकारी टाइप करें। Changed fields yellow highlight होंगी।
4
SAVE करें
नीचे "SAVE" button दबाएं। Data automatically Supabase cloud में save हो जाएगा।
💡 ध्यान रखें:
• JD/DEO केवल अपने अधिकार क्षेत्र के records edit कर सकते हैं
• Save करने के बाद record history में change दर्ज हो जाता है
• Document (PDF/Image) Cloudinary पर upload किया जा सकता है

🔍 Search और Filter कैसे करें

🔎 Quick Search
Toolbar में search box में कुछ भी टाइप करें — Name, Unique ID, District — तुरंत result दिखेगा।
🔽 Column-wise Filter
Table के header के नीचे हर column में एक filter box है। उसमें टाइप करने पर सिर्फ matching rows दिखेंगी।
🔄 Reset Filters
Toolbar में "RESET FILTERS" button दबाएं — सभी filters हट जाएंगे और पूरी list फिर दिखेगी।
📊 Advanced Filter
"OTHER TOOLBAR" → "Advanced Filter" → Category, Gender, District आदि से filter करें।

🖨️ Print और Export कैसे करें

📄 Single Record PDF
Record खोलें → Form में "PRINT PDF" button दबाएं → उस शिक्षक की वरीयता सूची PDF बनेगी।
📊 List PDF Print
Toolbar में "PRINT PDF" button → filtered list का PDF बनेगा। पहले filter लगाएं फिर print करें।
📥 Excel Export
"OTHER TOOLBAR" → "Export Excel" → current filtered data Excel file में download होगा।
📋 Summary Print
"SUMMARY" button → summary page खुलेगा → वहाँ से PDF या print कर सकते हैं।

⚙️ Admin Panel (केवल DPI)

⚠️ यह section केवल DPI admin के लिए है
🔑 Passwords Manage
DPI Panel → "Passwords" tab → सभी JD/DEO के passwords देखें, बदलें या reset करें।
🔧 Maintenance Mode
DPI Panel → Maintenance tab → ON करें। JD/DEO को maintenance screen दिखेगी। काम पूरा होने पर OFF करें।
📤 Data Import (Excel)
"OTHER TOOLBAR" → "Import Excel" → Excel से bulk data Supabase में upload करें।
📅 Access Period Set करना
DPI Panel → Settings tab → Global/Per-user access dates set करें। Deadline सभी users को दिखेगी।
